This is a site to check what the in-game time (IGT) threshold is for an Okami speedrun submission to require video proof, as determined by the Okami speedrunning community.

Naturally I need JS to run this page, to retrieve the current world record times for every speedrunning category (and calculate the threshold time (it's WR * 1.1 btw)). If you're still not comfortable with allowing JS here, you'll have to head to the speedrun.com page, check the WR times, and do the calculations yourself (which of course requires JS as well there).

If you're interested in speedrunning the game tho be sure to check out our Wiki and our Discord! (All links are on the SRC page too.)
